# BlockWorld - Мобильная аркадная игра

🎮 **BlockWorld** - это аркадная игра в стиле Minecraft для мобильных устройств и ПК, разработанная на чистом HTML5, CSS3 и JavaScript.

## 🎯 Особенности

✅ **Мобильная оптимизация** - адаптивный интерфейс для всех устройств
✅ **Простые управление** - кнопки для движения и действий
✅ **Сбор блоков** - добывайте различные типы блоков
✅ **Строительство** - воздействуйте с предметами из инвентаря
✅ **Динамический мир** - случайно генерируемая местность
✅ **Система здоровья** - избегайте урона и останавливайте врагов
✅ **Инвентарь** - управляйте собранными блоками

## 🚀 Быстрый старт

1. Откройте `index.html` в веб-браузере
2. Используйте кнопки управления для движения
3. Копайте блоки кнопкой "Копать"
4. Строите блоки кнопкой "Строить"

## 🎮 Управление

### Клавиатура
- **Стрелки** или **WASD** - движение
- **Пробел** - прыжок
- **J** - копать
- **B** - строить

### Мобильные кнопки
- **⬆️⬇️⬅️➡️** - движение
- **⛏️ Копать** - добыча блоков
- **🧱 Строить** - постройка блоков
- **🚀 Прыгать** - прыжок

## 🎨 Блоки

- 🟫 **Земля (Dirt)** - базовый материал
- ⬜ **Камень (Stone)** - прочный материал
- 🟪 **Дерево (Wood)** - строительный материал
- 🟩 **Листья (Leaves)** - декоративный материал

## 📱 Адаптивность

Игра оптимизирована для:
- Мобильные телефоны (320px+)
- Планшеты (768px+)
- ПК мониторы (1920px+)

## 📊 Система очков

Каждый собранный блок добавляет +1 к счету. Попытайтесь собрать как можно больше блоков!

## ⚙️ Технологии

- **HTML5 Canvas** - 2D графика
- **Vanilla JavaScript** - игровая логика
- **CSS3** - адаптивный дизайн
